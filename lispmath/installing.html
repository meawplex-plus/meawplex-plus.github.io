Installing lispmath

To install lispmath, simply install it from Ultralisp:
Install Quicklisp, then eval the following in your Lisp REPL:


	(ql-dist:install-dist "http://dist.ultralisp.org/"
                      :prompt nil)
Or, for Qlot users, add the following to your qlfile:

	dist ultralisp http://dist.ultralisp.org/
	ql :all :latest
	ultralisp :all :latest

Then, use the command (ql:quickload "lispmath") to load lispmath. And you're done!
